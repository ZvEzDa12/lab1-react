# Счетчик калорий

Мини-приложение для расчёта суточной нормы калорий (лабораторная работа №1).

## Технологии

- **Vite** — сборщик
- **React** + **TypeScript**
- Стили из макета (классы `form__control`, `counter-result_active` и др.)

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который выведет Vite (обычно http://localhost:5173).

## Сборка

```bash
npm run build
```

Готовые файлы появятся в папке `dist`.

## Реализованный функционал

- Форма: пол, возраст, рост, вес, физическая активность
- Валидация в реальном времени: возраст 0–150, рост и вес ≥ 0; при ошибке — класс `form__control_error` и сообщение в `span.form__error`
- Ограничение ввода: возраст при < 0 или > 150 подставляется 0/150; рост и вес при < 0 подставляется 0
- Кнопка «Рассчитать» отключена (`disabled`), пока форма невалидна
- Кнопка «Очистить поля» сбрасывает все поля к начальным значениям
- Расчёт по формулам Миффлина — Сан-Жеора и вывод двух норм: для поддержания жизнедеятельности и для поддержания веса (с учётом коэффициента активности)
- Блок результата изначально скрыт и показывается (класс `counter-result_active`) после нажатия «Рассчитать»
